rotateZ(βdeg) 矩阵推导

x y
r(x, y)
(x`, y`) α β 

// 单位圆得到等式
x` = cos(β+α) * r
y` = sin(β+α) * r
z` = z
                    

// 已知
r = √(x^2+y^2)
// 和差化积公式
sin(β+α) = sin(β)cos(α) + cos(β)sin(α)
cos(β+α) = cos(β)cos(α) - sin(β)sin(α)
// 正弦余弦定义
cos(α) = x/r
sin(α) = y/r
                    

// 愉快的代入 
x` = (cos(β)x/r - sin(β)y/r) * r = cos(β)x - sin(β)y
y` = (sin(β)x/r + cos(β)y/r) * r = sin(β)x + cos(β)y
                    

/* 所以猜想旋转矩阵 ＝> */ |cos(β) -sin(β)|
                        |sin(β)  cos(β)|

/* 验证下     => */      |x`|   |cos(β) -sin(β)|   |x|
                        |y`| = |sin(β)  cos(β)| * |y|

x` = cos(β)x - sin(β)y
y` = sin(β)x + cos(β)y
